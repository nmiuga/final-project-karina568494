# prompt.md

Hi! I'm in a beginning app development course. I have been given instruction for a project and my professor has asked that we start with AI help and then customize the app to our liking. Could you create an app with simple-to-customize code and a user-friendly layout that uses the Google Books public API?
I would like for a user to be able to create a digital library with multiple bookshelves to sort their unread books, read booka, currently reading, and favorites. I would like to be able to search up books and add books manually to the shelves.
I would also like for the app to be able to start a 'reading session' that provides a distraction-free timer and a place for the user to track what pages they read. In addition, I would like a journal aspect to the app where the user is prompted to reflect over the specific pages they read. These sessions and reflections would be stored in a reading log section of the app.
